# Contributing to AI Agent Workflow Automation Demo

Thank you for your interest in contributing! This project is designed to be a practical, production-style reference for AI Agent automation.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Set up your environment**:
   ```bash
   make install
   ```
3. **Run tests** before submitting:
   ```bash
   make test
   ```
4. **Follow code style**:
   ```bash
   make lint
   make type
   ```
5. **Submit a Pull Request** with a clear description of the changes.

## Development Guidelines

- **Write tests** for every new feature or bug fix.
- **Keep the demo runnable** — the MiniERP app should always start with `make start`.
- **Document changes** in the README if they affect the user-facing API or behavior.
- **One feature per PR** — avoid bundling unrelated changes.

## Code of Conduct

- Be respectful and constructive.
- Focus on what is best for the community and the project.
- Show empathy towards other contributors.

## Questions?

Open an issue or start a discussion. We're happy to help!
