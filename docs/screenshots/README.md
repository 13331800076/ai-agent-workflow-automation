# Screenshots

Place your demo screenshots here for the README.

## How to generate

1. Run the CLI demo:
   ```bash
   make cli-demo
   ```

2. Screenshot the artifacts directory:
   ```bash
   ls artifacts/task_*/screenshots/
   ```

3. Copy the best screenshots to this folder and update the README.

## Suggested screenshots

- `dashboard.png` — MiniERP home page
- `trace.png` — CLI execution trace with colored output
- `artifacts.png` — Artifact directory structure
